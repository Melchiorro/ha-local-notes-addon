# Home Notes Add-on for Home Assistant

A simple note-taking tool for your household. Keep grocery lists, home maintenance tasks, or shared family reminders directly in your Home Assistant.

## Why this add-on?
* **Shared Access:** Anyone with access to your Home Assistant can view and edit notes.
* **Image Attachments:** Take a photo of a broken part, a serial number, or a shopping item and attach it to a note.

## Installation
1. Go to **Settings** -> **Add-ons** -> **Add-on Store**.
2. Click the **three dots** (top right) -> **Repositories**.
3. Paste the repository URL: `https://github.com/Melchiorro/ha-local-notes-addon`
4. Find **My local notes** in the list and click **Install**.
5. Start the add-on and enable **Show in sidebar**.

## Configuration
The add-on uses port `15026` by default for Ingress communication. You can change the logging level in the configuration tab if needed.
